# Configuration du .env
(se baser sur le .env.example)
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=atelier
DB_USERNAME=root
DB_PASSWORD=
```
